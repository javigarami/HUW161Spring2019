This is my font selection. I chose Lora for the secondary headings and other text to create a contrast between the company's main header which is a semi connected script style of typography and the typeface chosen for larger text blocks. This promotes easy reading on the website and allows the brand to express itself using the Sacramento typeface as an accent.

Lora is a well-balanced contemporary serif with roots in calligraphy. It is a text typeface with moderate contrast well suited for body text.
